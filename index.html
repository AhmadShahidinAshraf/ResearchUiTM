SOAL SELIDIK UNTUK PENYELIDIKAN

YBhg. Datuk/ Dato'/ Datin/ Dr./ Sir/ Madam,

 Saya merupakan seorang pelajar Ijazah Sarjana dalam Pengurusan Sistem Pejabat di Universiti Teknologi MARA.

Soal selidik ini direka untuk pengumpulan data mengenai sejauh mana kakitangan di Ibu Pejabat Kementerian Kesihatan Malaysia (IPKKM) menerima dan menyesuaikan diri dengan perubahan teknologi dalam pelaksanaan tugas harian mereka. Kajian ini menggunakan teori Technology Acceptance Model (TAM) untuk mengukur penerimaan kakitangan di IPKKM terhadap teknologi baharu. Oleh itu, saya menggunakan sistem eTAMS sebagai alat ukur atau instrumen untuk menilai keberkesanan penerimaan teknologi baru dalam pelaksaan tugas harian kakitangan di IPKKM secara menyeluruh.

Anda telah dipilih untuk menyertai kajian ini dengan mengisi borang soal selidik ini. Indentiti peribadi dan jabatan anda akan dirahsiakan, dan maklum balas anda juga akan dirahsiakan.

Terima kasih atas kerjasama anda.
